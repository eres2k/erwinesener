## 💻 Local Development

### Prerequisites
- Modern web browser
- Optional: Local web server (for development)

### Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/eres2k/erwinesener.git
   cd erwinesener
   ```

2. **Open in browser:**
   - Simply open `index.html` in your browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8000

     # Or Node.js
     npx serve
     ```

3. **View the site:**
   - Navigate to `http://localhost:8000`
