# Erwin Esener | Personal Portfolio

A personal portfolio website showcasing professional projects, visual creations, and music remixes by Erwin Esener - WHS Manager Austria at Amazon Logistics.

**Live Site:** [https://eres2k.github.io/erwinesener/](https://eres2k.github.io/erwinesener/)

## Featured Projects

### 1. safety-analytics
**Enterprise Workplace Safety Intelligence System**

Comprehensive 33,641-line enterprise-grade safety intelligence platform for Amazon WHS Austria featuring revolutionary 4-tier progressive disclosure for GDPR/HIPAA compliance.

- 4-tier privacy system
- Interactive charts and visualizations
- PDF export functionality
- Zero-error calculations

**Tech:** Vanilla JS, Chart.js, PapaParse, jsPDF

[Launch Application](https://safety-analytics.netlify.app)

### 2. safety-hub
**Mobile-First Progressive Web Application for Safety Auditing**

Next-generation PWA revolutionizing workplace safety auditing with offline-first architecture, real-time analytics, digital signatures, and photo capture.

- Offline-first architecture
- Photo capture capabilities
- Auth0 enterprise security
- Real-time analytics

**Tech:** React 18, TypeScript, Vite, TailwindCSS, Supabase, Auth0

[Launch Application](https://safety-hub.netlify.app)

### 3. AMZL WHS Quest
**Retro 16-Bit Warehouse Safety Educational Game**

Innovative educational game transforming OSHA compliance training into an engaging retro gaming experience with procedurally generated 16-bit aesthetics.

- 16-bit retro graphics
- 12 chiptune tracks
- 9 boss battles
- 100+ real warehouse hazard scenarios

**Tech:** JavaScript, HTML5 Canvas, Web Audio API, PWA, Cordova

[Launch Game](https://amzlwhsquest.netlify.app)

### 4. Healthy Brunch Club
**Progressive Web App for Bio-Brunch Restaurant**

Modern PWA for a Vienna bio-brunch restaurant combining static site speed with dynamic content management, filterable menus, and online reservations.

- Dynamic menu system
- Offline capability
- Git-based CMS
- Serverless functions

**Tech:** HTML5, CSS3, Vanilla JS, Netlify Functions, Netlify CMS, PWA

[View Live Site](https://healthybrunchclub.at)

## Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/eres2k/erwinesener.git
   cd erwinesener
   ```

2. **Open in browser:**
   - Simply open `index.html` in your browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8000

     # Or Node.js
     npx serve
     ```

3. **View the site:**
   - Navigate to `http://localhost:8000`
