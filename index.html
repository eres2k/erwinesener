AMZL WHS QUEST

Retro 16-Bit Warehouse Safety Educational Game

An innovative educational game that transforms workplace safety training into an engaging retro gaming experience. Built entirely with vanilla JavaScript and procedural generation, QUEST merges nostalgic 16-bit aesthetics with cutting-edge web technologies, making OSHA compliance genuinely fun through epic boss battles and character progression.

🎮

Authentic Retro Gaming

True 16-bit aesthetic with 256×224 SNES resolution, CRT effects, 12 chiptune tracks, and procedurally generated pixel art

👥

Five Playable Characters

Unique warehouse coordinators with distinct abilities: The 5S Monk, The Oracle, The Siren, The Ghost, and The Manager

🏭

100+ Real Hazards

Authentic warehouse safety scenarios including PIT violations, ergonomic issues, fire safety, and OSHA regulations

👾

Epic Boss Battles

Nine unique bosses from Labour Inspector to Mecha Jeff Bezos, each with distinct attack patterns and personalities

🎨

Zero-Dependency Architecture

100% procedurally generated assets with vanilla JavaScript—no frameworks, libraries, or external image files

📱

Cross-Platform PWA

Progressive Web App installable on desktop/mobile with Apache Cordova for native Android deployment

SAFETY-analytics

Enterprise Workplace Safety Intelligence System

A comprehensive enterprise-grade workplace safety intelligence system for Amazon WHS Austria. With 33,641 lines of pure vanilla JavaScript, this sophisticated web application transforms raw incident data into actionable insights while maintaining the highest standards of privacy compliance (GDPR/HIPAA) through a revolutionary 4-tier progressive disclosure model.

🔒

4-Tier Privacy Protection

Progressive disclosure model with 60+ protected fields aligned with GDPR Articles 5, 9, and 32—avoiding potential €20M fines

📊

Comprehensive Analytics

Overview dashboards, injury analytics, near-miss intelligence, inspections, weekly business reviews, and advanced reporting

✅

Zero-Error Calculations

6 custom utility functions eliminate all NaN/Infinity errors with bulletproof data validation and type safety

📈

Advanced Visualizations

Interactive Chart.js dashboards with real-time updates, trend analysis, heat maps, and severity distribution matrices

📄

Multi-Format Export

PDF generation with charts and privacy-aware Excel export for compliance documentation and regulatory submissions

💯

Vanilla JavaScript Excellence

33,000+ lines of pure JavaScript without framework dependencies—demonstrating mastery of fundamental web technologies

03

safety-hub

Mobile-First Progressive Web Application for Safety Auditing

A next-generation Progressive Web Application revolutionizing workplace safety auditing and compliance management for Amazon Logistics warehouse operations. Built with offline-first architecture, Safety Hub enables safety coordinators to conduct comprehensive inspections directly from mobile devices, even without internet connectivity.

📱

Offline-First PWA

Complete audit workflows available without network connectivity using service workers and IndexedDB caching

✅

Digital Audit System

Replace paper checklists with dynamic questionnaires, photo capture, and digital signatures - 70% faster than traditional methods

📊

Real-Time Analytics

Visual dashboards with trend analysis, pass/fail rates, and top findings across multiple sites

🚨

Action Tracking

Assign corrective actions with due dates, automated reminders, and before/after photo documentation

🔐

Enterprise Security

Auth0 integration with OAuth 2.0, role-based access control (RBAC), and JWT token security

📄

Professional Reporting

Automated PDF export with charts, photos, signatures, and Excel data export for compliance documentation

Technology Stack

React 18 TypeScript Vite TailwindCSS Supabase Auth0 Capacitor Netlify
